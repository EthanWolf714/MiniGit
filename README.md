## MiniGit
This is a mini git clone i wanted to work on as a resume project.


# Project Structure
minigit/
├── src/
│   └── MiniGit.java        # Everything in one file to start!
├── .minigit/               # Created when you run 'init'
│   ├── objects/            # Stores file snapshots
│   └── HEAD                # Current commit hash
└── README.md


Commands so far:
- minigit init
- minigit add <file>
